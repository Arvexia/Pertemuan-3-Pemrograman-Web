## 📝 Deskripsi Tugas
Tugas ini merupakan bagian dari modul praktikum pertemuan pertama yang bertujuan untuk memahami cara menyusun struktur halaman web dan menambahkan konten menggunakan bahasa markah (HTML). Halaman ini menampilkan informasi mengenai **AMIKOM Creative Economy Park**.

### Fitur & Tag yang Digunakan:
* **Heading (`<h1>` - `<h3>`):** Digunakan untuk judul utama, visi, dan tagline.
* **Image (`<img>`):** Menampilkan logo Amikom ke dalam halaman web.
* **Link (`<a>`):** Menambahkan tautan eksternal menuju website Universitas Amikom Yogyakarta.
* **Format Teks:** Penggunaan teks tebal (`<b>`), garis bawah (`<u>`), dan garis horizontal (`<hr>`).
* **Blockquote:** Digunakan untuk menampilkan tagline khusus secara terstruktur.

## 📂 Struktur File

* `logo amikom.png`: Asset gambar yang digunakan dalam halaman web.

## 🛠️ Cara Menjalankan
1.  Buka aplikasi **Visual Studio Code (VSCode)**.
2.  Clone atau unduh repositori ini.
3.  Buka file `Challenge_24.11.6282.html` di VSCode.
4.  Gunakan plugin **Live Server** untuk melihat hasil desain secara langsung di browser.
